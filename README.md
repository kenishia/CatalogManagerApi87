# zOS Connect OpenAPI3 Project

## Overview
Catalog Manager API created using 3.0.87 image and pushed to this repo

The OpenAPI specification is available at:

```
http://localhost:9080/openapi
```

A UI for this OpenAPI Specification is available at:

```
http://localhost:9080/openapi/ui
```

Github remote name is the org name

Combination of github and local runners are used for github actions.  local runner label is kc-localwin
